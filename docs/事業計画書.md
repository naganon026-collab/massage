# Logic Post 事業計画書

## 1. 事業概要

### 1.1 サービス名
**Logic Post**（ロジックポスト）

### 1.2 サービス説明
マッサージ・ヘッドスパ・整体・美容室などの店舗向けに、SNS・ブログ用のSEOテキストをAIで生成するアシスタントツール。フォームに3つの質問に答えるだけで、AIがお店の情報・文体・天気まで反映した投稿文を**5媒体分**まとめて生成する。

### 1.3 ターゲット顧客
- 美容室・ヘッドスパ・ネイルサロン
- マッサージ・整体・整骨院
- その他地域密着型の小規模店舗（SNS運用が必要な事業者）

### 1.4 解決する課題
- 媒体ごとに書き直す手間（Instagram・Google・ブログ・LINEで文体が違う）
- SNS投稿のネタ切れ・継続の難しさ
- 専門家としての情報発信のハードルの高さ
- AIで試したが「いまいちだった」という経験（→ 過去投稿を学習させてお店らしい文体で生成）

## 2. 製品・サービス詳細

### 2.1 出力媒体（5媒体）
| 媒体 | 説明 |
|------|------|
| Instagram | 投稿文・キャプション |
| Google Map投稿（GBP） | クチコミ返信・投稿 |
| ブログ | SEO記事・FAQ形式（プロプラン限定） |
| LINE | 公式アカウント配信用メッセージ |
| ショート動画 | Reels/TikTok/Shorts用台本 |

### 2.2 投稿パターン（9種類）

#### 集客・認知
- **パターンA：ビフォーアフター** — 施術の変化を通じて「私も予約したい」を生む
- **パターンB：プロの知識・教育コンテンツ** — 保存・シェアを生む有益情報
- **パターンD：お客様の喜びの声** — 第三者の言葉で信頼と安心感を伝える

#### つながり・ファン化
- **パターンE：スタッフの人柄・指名獲得** — 「この人に任せたい」を育てる
- **パターンH：サロンの裏側・日常** — 清潔感・丁寧さ・人間味を伝える

#### 今すぐ動いてほしい
- **パターンC：お知らせ・限定情報** — タイムリーな情報で即時来店・予約を促す
- **パターンG：コメント・クチコミへの返信** — SNS/Googleに届いたコメントへの誠実な返信

#### 手軽・ネタに困ったとき
- **パターンF：今日の旬な投稿** — 日付・季節・時期をもとにAIが自動で最適な内容を生成
- **パターンI：画像から生成** — 画像アップロードで画像に合った投稿文を自動作成

### 2.3 主な機能
- **5媒体一括生成** — 一度の入力で5媒体分を同時作成
- **画像から生成** — 施術写真・メニュー画像をアップロードして投稿文を自動作成
- **文体学習** — 過去の投稿を登録するとトーンや言い回しを学習し、お店らしい自然な文章を継続
- **ニュース連動** — 業種に合った最新ニュースを選ぶと専門家目線のコメント付き投稿を自動生成
- **URLからの情報取得** — 店舗サイトURLを入力すると基本情報を自動抽出
- **改善・再生成** — 「もっと短く」「カジュアルに」「共感を強く」など指示で微調整

### 2.4 外部連携
- **LINE** — LINE公式アカウントへの直接投稿
- **Late** — Instagram・Google Map（GBP）への投稿（運営者がLateと契約しAPIキーを共有）
- **Open-Meteo** — 天気情報の自動取得（季節感・気候に合った文章用）
- **ニュースAPI** — 業種に合った最新ニュースの取得
