# API 料金見積もり（Gemini 2.5 Flash）

## 運用見積もり（1回 $0.04 で計算）

| 前提 | 1回あたり | 月150回（1日5回×30日） |
|------|------------|-------------------------|
| **単価** | **$0.04** | **$6**（約 900 円・1ドル=150円） |

- 1日5回までに制限しているため、月あたり最大 150 回。
- 売価 2,980 円/月の場合、1ユーザーが月150回使っても API 約 $6 で収まる想定。

---

## 使用モデル

- **`gemini-2.5-flash`**（`src/app/api/generate/route.ts` ほか）

## 公式料金（Google AI Studio・Paid Tier）

出典: [Gemini API Pricing](https://ai.google.dev/gemini-api/docs/pricing) — Gemini 2.5 Flash

| 項目 | 料金（1M tokens あたり USD） |
|------|------------------------------|
| **Input**（テキスト/画像/動画） | **$0.30** |
| **Output**（思考トークン含む） | **$2.50** |

- 無料枠あり（Free Tier）。本見積もりは **Paid Tier** を前提。

---

## 1回の「生成」あたりの想定トークン

このアプリの **1回の生成**（`/api/generate` 1リクエスト）では **1回だけ** `generateContent` を呼びます。

| 種類 | 控えめ | 標準的 | 多め（スクレイプ長文・全媒体） |
|------|--------|--------|--------------------------------|
| **Input** | 25,000 | 40,000 | 60,000 |
| **Output**（本文＋思考トークン含む） | 5,000 | 8,000 | 12,000 |

- Input: システムプロンプト・媒体別指示・店舗情報・スクレイプ本文・ニュース・季節コンテキストなど。スクレイプが長いと増える。
- Output: JSON（Instagram / GBP / ポータル / LINE / ショート台本など）。Gemini 2.5 Flash は思考トークンも課金対象のため、実際の出力より多めになりうる。

## 「1日5回 × 30日」＝ 月150回の場合

| ケース | 1回あたり | 月150回の合計（USD） | 概算（円・1ドル=150円） |
|--------|------------|----------------------|---------------------------|
| 控えめ | 約 $0.02 | **約 $3** | 約 450 円 |
| 標準的 | 約 $0.032 | **約 $4.80** | 約 720 円 |
| 多め | 約 $0.048 | **約 $7.20** | 約 1,080 円 |

**目安: 月 約 $4〜7（約 600〜1,050 円）**  
（「生成」ボタンのみの利用で、他APIを使わない場合）

## 補足

1. **実際の $2.96/50回**  
   50回で $2.96 だと 1回あたり **約 $0.059** です。上記の「生成だけ」の見積もりより高めになる要因の例：
   - **思考トークン**が多く出ている（Output が 15k〜20k 程度）
   - **他APIも利用**している（URL読み込み時の `extract-info`、店舗分析の `analyze-shop`、画像生成の `generate-image`、リファインの `generate-refine` など）
   - **プロンプトが長い**（スクレイプ本文・サンプル文を多く入れている）

2. **他の Gemini 呼び出し**  
   - `extract-info`: テキスト最大 20k 文字 → 1回あたり数セント程度  
   - `analyze-shop`: スクレイプ本文を送るため、長文だと 1回あたり数セント〜十数セント  
   - `generate-refine`: 1回あたり generate と同程度〜やや少なめ  
   これらが「1日5回の生成」に加えて使われていると、月額は上記目安より増えます。

3. **正確な請求額**  
   [Google AI Studio の利用状況・請求](https://aistudio.google.com/) で、入力/出力トークン数と請求額を確認できます。

## まとめ（1日5回・30日の場合）

| 前提 | 月額（USD）目安 |
|------|------------------|
| **1回 $0.04 で見積もり（運用目安）** | **$6**（150回） |
| 生成のみ・トークン控えめ（公式単価ベース） | 約 **$3** |
| 生成のみ・トークン標準 | 約 **$5** |
| 生成のみ・トークン多め or 思考多め | 約 **$7** |
| リファイン・URL取得・分析などをよく使う | 上記に **+$1〜3** 程度 |

- **運用計画では 1回 $0.04 で計算**し、月150回で **$6** を目安にすると安全側の見積もりになります。
- 実際の請求は利用パターンとトークン量で変わるため、最初の1〜2ヶ月はダッシュボードで実績を確認するのがおすすめです。
